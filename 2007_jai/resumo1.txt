Título: Script para automatização de envio de e-mails (não lembro do título)

OBJETIVOS (100 a 300 caracteres )
Desenvolver um script que avise automaticamente a mudança da responsabilidade de atualização da página do Curso de Ciência da Computação e informe o responsável anterior que sua quinzena de atualização terminou. 

Materiais e Métodos ( 100 e 700 caracteres )
Para o desenvolvimento desse script foi utilizada a linguagem de programação PHP e shell script, que vem a ser um, arquivo que armazena linhas de comando. O script desenvolvido em PHP, ao ser executado, verifica a dada corrente gerando o índice que determinara o atual e o próximo responsável, enviando um e-mail com um texto pré formatado, para cada um destes. O arquivo shell script é responsável por chamar a execução do script email.php. Para que essa execução ocorresse quinzenalmente, o arquivo shell script foi adicionado ao crontab, e este foi configurado para realizar uma execução do script todos somente dias 1 e 15 de cada mês as 8 horas. 

Resultados ( 100 a 700 caracteres )
A utilização do script desenvolvido automatizou o envio de e-mail, evitando um trabalho manual. O mesmo serviu também para evitar possíveis falhas, como  esquecimento, tanto de envio de e-mail como de atualização da página. Pois como aprimoramento do script, foi acrescentado o envio de uma mensagem de aviso no intervalo da quinzena, para relembrar o responsável de seu compromisso. 

Conclusão ( 100 a 300 caracteres )
O desenvolvimento de um script fácil de automatização, possibilitou a melhora no desenvolvimento das tarefas do grupo devido sua automatização. Como trabalhos futuros, pretende-se estender o aviso automatizados para demais tarefas periódicas do grupo.
